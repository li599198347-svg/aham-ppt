# Aham Design System（v6.1 · PPT/Office 轨）

> **北极星**：极简、克制、留白、内容优先的桌面 AI 气质。性格：**冷色的纸、克制的金属感、对话式**。
> 取值唯一事实源：`tokens.css`（镜像自 aham-ui/tokens.json）。本技能只产出 PPT，属 **Office 轨 · PPT 子模型**，落地细则见 `aham-ui-office.md`。

**Brand**: Aham（示例品牌 · Demo Brand）
**Locale**: 主要场景 zh-CN（简体中文），品牌名用拉丁字母。
**Applicability**: 严肃 B2B；严谨、克制、工程化、可靠、懂行。
**Brand attributes**: 严谨 (Rigor) · 克制 (Restraint) · 工程化 (Engineered) · 可靠 (Dependable) · 懂行 (Domain-aware)

> 📘 Aham 是虚构示例品牌。替换品牌名/主色/字体即可适配你自己的品牌，见根目录 `README.md`。

## 落地轨道（按介质分四轨）

Aham UI 按落地介质分四轨：**网页 / 应用(macOS) / Office / 邮件**，布局心智不同。
**本技能 = Office 轨 · PPT 子模型**：固定画布 16:9、定宽定位、母版网格；统一单蓝 + 冷色纸。

## 1. 内容基本功 CONTENT FUNDAMENTALS

**Voice**: 一位 15 年经验的资深顾问在说话——专业、直接、克制，用数字不用形容词。**绝不**亲昵或俏皮。

| 渠道 | 语气 |
|---|---|
| 方案正文 | 严谨第三人称——"调研显示核心计划模块从未正式启用。" |
| 执行摘要 | 决断、建议先行——"我们建议分三期实施新系统。" |
| 客户邮件 | 礼貌、简短、行动导向——"已将修订版发您，请于周五前反馈。" |
| 合同 / 报价 | 法律口吻、零修饰——"本报价有效期 30 日。" |
| 软件 UI 文案 | 短、祈使、无感叹号——"保存"、"确认删除该工单？" |
| 错误信息 | 原因 + 补救——"网络中断，请检查连接后重试。" |

**人称**：第三人称为主；"我们"仅用于执行摘要/建议。绝不用"我"。
**字形**：品牌名恒为 `Aham`。UI 标签 ≤ 4 字；按钮 ≤ 6 字。
**中西混排空格**：中文与拉丁/数字间留半角空格（`使用 A 系统`、`上线 45 天`）；数字与单位不留空格（`30%`、`¥1,200`）。

**禁用词**（交付物绝不出现）：
> 赋能 · 颠覆 · 生态 · 闭环 · 最佳实践 · 全链路 · 一站式 · 数字孪生 · 智慧 XX · AI+ · 元宇宙 · 无缝 · 极致 · 唯一 · 行业第一 · 显著 · 大幅 · 明显 · 考虑 · 可能 · 或许 · 加强 · 完善 · 优化 · 积极推进

一律换成**具体量化**：不写"显著提升"写"提升 37%"；不写"全面赋能"写"覆盖计划、执行、追溯 8 个模块"。

**Emoji**：❌ 绝不。状态用符号+文字（`● 合格` / `▲ 预警` / `✕ 异常`）。
**数字格式**：货币 `¥1,200,000`；百分比一位小数 `37.5%`；区间 `15%–20%`；日期 `YYYY-MM-DD`；时间 `HH:mm`。**所有数字用等宽 mono**（`JetBrains Mono` / `Consolas`），每个数字标注来源 + 时间。
**建议四要素**：【具体行动】+【量化目标】+【责任方】+【时间节点】。
**Vibe**：安静的权威。白皮书，不是落地页。读者应觉得在读一份麦肯锡级交付物。

## 2. 配色体系（颜色唯一来源，查表执行）

**克制色板：唯一色相 = 蓝，其余皆中性灰 + 语义色。** 底永远纯白。

### 2.1 主色 / 表面
- **三层表面**：`#FFFFFF` 内容/底 · `#F3F3F3` 面板(极少) · `#E7E7E7` 线/分隔/选中。深度靠**层差**，不靠材质/模糊/阴影。
- **强调蓝**：`#336EE8`（hover `#5C8BED` / press `#164EC3`）。**唯一色相，只点缀**：品牌点 / 主操作 / 选中下划线 / 推荐标 / **单个**高亮数据点 / 封面·关键页短装饰条。**蓝若第一眼注意到就超标，绝不铺底。**

### 2.2 中性墨（四级）
`#262626` 主（正文/标题/数字）· `#6E6E6E` 次（次要/表头副文）· `#9B9B9B` 三（占位/页脚/来源/轴标签）· `#C4C4C4` 四（最弱线/禁用，不承载文字）。**绝不用纯黑 `#000`。**

### 2.3 语义色（极弱，仅真风险，**必须颜色+符号/文字双通道**）
成功 `#5A7A60` · 警示 `#8A7333` · 风险 `#9E3D31`（浅底 `#F0F2F0` / `#F4F1E9` / `#F4ECEA`）。
**硬要求**：状态 = **6px 点 + 文字**，颜色之外文字本身可区分（达标/预警/风险），去色后仍可辨。**禁红黄绿灯、禁 pill/徽章/色块、禁整行整列铺色。**

### 2.4 图表色顺序
数据元素**灰阶** `#9B9B9B`（主）/ `#C8C8C8`（次），**至多一个 `#336EE8`** 高亮当前/重点项；坐标轴 `#E7E7E7`、标签 `#6E6E6E`。绝不多色，无饼图/环形图，阈值只染文字。

## 3. 字体体系

**单一无衬线，禁衬线。层级靠字号 + 字重，不靠颜色、不靠衬线。**
- **无衬线（标题/正文/UI）**：`Inter, 'PingFang SC', 'Microsoft YaHei', sans-serif`（CJK = 微软雅黑/PingFang）。
- **等宽 mono（所有数字、编号、时间、ID）**：`'JetBrains Mono', Consolas, monospace`。**CJK 禁入等宽。**
- 字重：标题 700/600、正文 400、强调 500/600；**不用 100–300 细体**。

## 4. 视觉基础 VISUAL FOUNDATIONS

**设计哲学（三支柱）**：① 清晰——内容是主角；② 克制——默认"不加"，每个颜色/装饰都要有理由；③ 层次——靠**三层背景层差**（纯色），不靠材质/模糊/阴影。

- **卡片**：无边框无阴影。成块才用 `#F3F3F3` 面板（圆角 12，少用），否则**留白 + `#E7E7E7` 细横线**分隔。选中/强调 = 扁平灰，**非蓝**。
- **表格**：只横线（`#E7E7E7`），**无竖线、无整行底色**；数字右对齐 mono；表头/合计靠**加粗**不靠色块。
- **间距**：4 基 `4/8/12/16/24/32/48/64/96`。**留白是首要分隔，先加间距再考虑线。**
- **圆角**：`xs4 / sm6 / md8(控件) / lg12(卡片) / xl16(modal)`；不滥用大圆角。
- **阴影**：FLAT——静置/hover 无阴影，仅浮层有（菜单/弹窗）。
- **背景**：白或近白；无渐变、无暖灰、无毛玻璃、无装饰纹理、无图片满铺。
- **动效**（屏幕端）：克制，默认 `0.18s`；无数字滚动/卡片浮动/光晕。
- **图标**：Lucide 线性单色 1.5–2px，**仅导航/发送栏**；内容区用 `•` + 文字。禁 emoji、多色/填充图标。
- **影像**：真实业务实景，低饱和专业调；绝不用握手/AI 生成的通用图。无图时用占位标 `[待替换·示例业务实景]`。

**绝对禁用**：渐变 · 3D · 投影(浮层除外) · 纯黑 `#000` · **衬线** · 第二装饰彩色 · 饼图 · 表格竖线 · emoji · 圆角滥用 · **蓝色铺底 / 整行整列铺色** · 红黄绿灯。

## 7. PPT / Office 规范（落地取值，见 `aham-ui-office.md`）

**尺寸**：16:9。**底永远纯白**，无渐变、无图片满铺、无色块墙。

### 7.1 母版 / Chrome
- 页眉：Section 路径（左，`#9B9B9B`）+ 品牌（右，`#262626` + 左侧 9×9 `#336EE8` 蓝点）+ `#E7E7E7` 细线。**无顶部蓝条。**
- Action Title：整句结论、无衬线 bold `#262626`（**无蓝竖线、无衬线**）+ `#E7E7E7` 细线。
- 页脚：`#E7E7E7` 细线 + 来源（左 `#9B9B9B`）+ 页码（右 mono `#9B9B9B`）。
- 关键页可在左下放 `#336EE8` 2pt 短装饰条（≈3cm），**全页唯一蓝面**。

### 7.2 字体映射（Office）
标题/正文 西文 `Inter`（无则 Calibri）· 中文 `微软雅黑`（备 SimHei）· 数字/编号 `Consolas`（↔ JetBrains Mono）。**标题不用衬线**；层级靠字号 + 加粗。**中文禁入等宽。**
PPT 字号：封面标题 32–40pt（左对齐，不居中堆叠）· 内容标题 24pt · 正文/要点 14–16pt 行距 1.3（要点 `•`+文字、编号 mono `01/02`）· 大数字 Consolas 44–60pt `#262626`（或一个关键数 `#336EE8`）。

### 7.5 图表（PPT）
柱/折线/环形——**灰阶 `#9B9B9B`/`#C8C8C8` + 一个 `#336EE8` 高亮当前项**，绝不多色。坐标轴 `#E7E7E7`，标签 `#6E6E6E`。无 3D、无阴影、无渐变、无饼图。阈值只染文字。
